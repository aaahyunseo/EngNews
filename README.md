# EngNews
**뉴스만 읽었을 뿐인데 영어와 트렌드가 쏙쏙?! 잉뉴스(EngNews)**

<br/>

## 프로젝트 개요

현대 사회에서 영어는 필수적이지만, 기존의 영어 학습 서비스들은 단순 암기와 회화에만 집중되어 있어 산업 현장의 실무자분들께는 실효성이 부족한 경우가 많습니다. 2024년 링글 보고서에 따르면, 국내 직장인의 77%가 업무 중 영어 실력 부족으로 인한 한계를 경험하신 것으로 나타났습니다. 특히 하루 평균 9.6시간을 근무하는 한국의 산업 현장에서, 급변하는 글로벌 환경에 대응하면서 별도의 영어 학습까지 병행하기란 현실적으로 어려움이 많습니다.

무역, 반도체 등 영어가 필수적인 산업 분야에서는 최신 동향 파악과 영어 학습을 동시에 해결할 수 있는 통합형 서비스가 절실히 필요합니다. 이에 저희는 산업별 특화 키워드를 기반으로 맞춤형 뉴스를 선별하고, 실시간 번역과 관련 용어 학습을 함께 제공하는 원스톱 통합 서비스를 기획하였습니다. 본 서비스는 산업 현장의 실무자분들께 업무 효율성과 영어 능력 향상을 동시에 지원하는 혁신적인 솔루션이 될 것입니다.

<br/>

## 역할

- **서버 개발**: 네이버API와 NYT API를 사용해 카테고리와 키워드 별 뉴스 기사를 가져오고, 크롤링을 통해 개별 기사를 불러오도록 설계하였습니다.
- **UI/UX 디자인**: 사용자가 쉽게 뉴스 기사에 접근하여 학습할 수 있도록 직관적인 인터페이스를 설계하였습니다.
- **AI 개발**: TensorFlow 및 PyTorch를 활용해 다양한 AI 모델을 학습하고 통합하였습니다. BERT를 사용해 문맥에 맞는 어휘 및 구문 분석을 수행하고, Transformer 기반 모델을 사용해 사용자가 요약한 내용을 피드백 하도록 설계하였습니다.

<br/>

## 핵심 기능

1. **맞춤형 뉴스 제공**
   - 다양한 News API를 사용해 실시간 뉴스 기사를 주제별로 분류하고, 사용자의 관심사에 맞춘 맞춤형 기사 제공합니다. 한국어와 영어 기사를 비교해 학습할 수 있습니다.

2. **AI 기반 어휘 및 문법 도구**
   - **BERT**와 **Google Cloud Natural Language API**를 활용해 뉴스 기사에서 어려운 단어나 구문을 실시간으로 분석하고 설명을 제공합니다.

3. **번역 연습**
   - 사용자는 한국어 기사를 영어로 번역하거나, 영어 기사를 한국어로 번역하는 연습을 할 수 있습니다. Transformer와 같은 모델을 활용하여 번역의 정확성과 자연스러움을 피드백하고, 더 나은 번역 방법을 제안해줍니다.

4. **듣기와 말하기 연습**
   - 뉴스 기사의 자연스러운 발음 오디오를 제공합니다.

## 타깃 이용자
- 영어 학습을 원하는 중급 이상의 실력자 / 영어 학습을 통해 실무 트렌드도 간편하게 얻고 싶은 직장인과 비즈니스 실무자

<br/>

## 차별성
- **실시간 뉴스 학습**: 실시간 뉴스로 흥미로운 주제와 최신 정보를 학습 가능하며, 국내외 뉴스를 비교해 문화적 맥락도 이해 가능.
- **산업 특화 학습**: 산업별 특화된 뉴스로 실무에 바로 활용 가능한 표현과 정보를 습득 가능.
- **AI 맞춤형 피드백**: 사용자의 관심사와 수준에 맞춘 뉴스 제공 및 실시간 영어 학습 피드백.
- **깔끔한 뉴스 영어 표현**: 기존의 영어 회화 앱과 달리 일상적인 대화가 아닌 뉴스 영어를 학습함으로써, 더 정제되고 깔끔한 영어 표현 학습 가능.

### Branch Naming Convention

| ------- | ---------------------------------- |
| main    | 서비스 브랜치                      |

| 머릿말   | 설명                                                 |
